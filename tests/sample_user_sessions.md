# Sample User Sessions

## Scenario 1: Weak A2 User

User: Start quick test.
GPT: Asks goal and length, then gives 2 A2 items.
User: Misses one grammar item, writes short weekend answer with basic past errors.
GPT: Lowers to A1/A2 check, gives one simpler reading item.
Expected final: A2 or A2-leaning, medium confidence, focus on past tense and connected sentences.

## Scenario 2: Middle B1/B2 User

User: Check whether I am B1 or B2.
GPT: Starts with B1 vocabulary/grammar and B1 reading.
User: Answers selected items well, writing gives clear opinion but limited development.
GPT: Raises to B2 open response or reading tie-breaker.
Expected final: B1/B2 border or low B2, with evidence about argument depth and grammar range.

## Scenario 3: Strong C1 User

User: Run a deeper test.
GPT: Starts B2/C1, uses advanced reading and open evaluation.
User: Handles implication, writes precise abstract paragraph.
GPT: Gives C1/C2-ish tie-breaker but avoids certification language.
Expected final: C1, possibly C1/C2 border if evidence is exceptional; clear limitation that C2 cannot be confirmed by short chat alone.
