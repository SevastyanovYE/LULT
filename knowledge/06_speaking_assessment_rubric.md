# Speaking Assessment Rubric

Retrieval keywords: speaking rubric, speaking assessment, audio, pronunciation, fluency, interaction, no audio, self-assessment, listening.

## Important Limitation

Without real audio, speaking assessment is limited. Text answers can show planning, grammar, vocabulary, and discourse, but not pronunciation, rhythm, stress, hesitation, or real-time repair.

## Criteria

- Fluency: Can the user continue speaking without excessive pauses?
- Pronunciation: Are sounds, stress, and intonation intelligible?
- Coherence: Are ideas organized and connected?
- Interaction: Can the user answer, ask, clarify, and respond naturally?
- Grammar: Are structures accurate enough for the message?
- Vocabulary: Is word choice accurate, varied, and topic-appropriate?
- Repair strategies: Can the user rephrase, correct, and recover when stuck?

## Level Signals

| Level | Speaking signal |
|---|---|
| A1 | Single phrases; heavy support needed |
| A2 | Simple routine exchanges; limited repair |
| B1 | Familiar conversation; pauses for vocabulary |
| B2 | Supported opinions; moderate fluency |
| C1 | Fluent, flexible, abstract topics, smooth repair |
| C2-ish | Natural, precise, idiomatic, strong interactional control |

## GPT Application

If audio is unavailable, label speaking as “not directly assessed” and ask optional self-assessment questions instead of assigning a strong speaking score.
